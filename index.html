Configurable Markup Application

This application loads a web scene and provides simple markup capabilities via the ArcGIS API for JavaScript 4.x. Add a FeatureLayer of points with Zs named "Feedback - Feedback" to your web scene and then append your web scene id to this url: ../index.html?webscene=your web scene id here
Please use this FeatureLayer as a template.
This application is built using:
  • Esri ArcGIS API for Javascript 4.x
  • Esri ApplicationBase  &  Configurable App Example
  • Esri Calcite Web Framework
